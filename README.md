## Overview

This project is a client-side password validation tool built with HTML, CSS, and vanilla JavaScript. It displays visual feedback as users type their password, helping them create secure passwords that meet specific criteria.

## Features

- **Real-time Validation**: Instant feedback as you type
- **Password Visibility Toggle**: Show/hide password with an eye icon
- **Character Count Display**: Shows current password length
- **Visual Feedback**: Color-coded validation indicators (red/green)
- **Five Validation Rules**:
  - Length greater than 5 characters
  - Length less than 10 characters
  - Contains at least one numerical character
  - Contains at least one special character
  - No spaces allowed

## Files

- `index.html` - Main HTML structure
- `style.css` - Styling and layout
- `script.js` - Password validation logic

## Installation

1. Clone or download the project files
2. Ensure all three files are in the same directory
3. Open `index.html` in a web browser

No build process or dependencies required!

## Usage

1. Open the application in your browser
2. Type a password in the input field
3. Watch as the validation indicators change color:
   - **Red**: Requirement not met
   - **Green**: Requirement satisfied
4. Click the eye icon to toggle password visibility
5. The character count updates automatically as you type
